# vue-trunquee

![Trunquee Demo](https://i.imgur.com/rggXkO7.gif)

A truncated marquee component for [Vue.js](http://vuejs.org/).

`Trunquee.vue` truncates text longer than its container, with a marquee reveal on mouse enter.
